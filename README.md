# Postgresql Admin (pgadmin) Web GUI

# Run (Recommended for easy start-up)
It will pop up X11 to display pgadmin in Firefox browser.
```
./run.sh

```

# Build
To build your local Docker images (instead of pulling automated pre-built image)
```
./build.sh
```
